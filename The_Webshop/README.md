## Informatie over het project
Dit project is een webshop voor bass gear: **The Low End**.

Belangrijkste onderdelen:
- Producten uit `pub/data/products.json`
- Productpagina met filters en sortering (`pub/browse.html`)
- Winkelwagen en bestellen (`pub/cart.html`)
- Adminpagina voor producten en bestellingen (`pub/admin.html`)
- Opslag van data in `localStorage`

## Waar het project online te vinden is
Live URL:
- `https://<jouw-site>.netlify.app`

## Instructies om het project lokaal te runnen
1. Open een terminal in de map `The_Webshop`.
2. Start een lokale server:
   - `npx serve pub`
3. Open de localhost-URL die in de terminal verschijnt.
4. Pagina's:
   - Home: `index.html`
   - Browse: `browse.html`
   - Cart: `cart.html`
   - Admin: `admin.html`
